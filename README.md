# Slang ESP32-IDF component
This component integrates the Slang programming language compiler with the ESP32-IDF framework, enabling developers to compile and use Slang in their ESP32 projects.
## Usage
To use the Slang ESP32-IDF component in your project, follow these steps:
**Clone the Repository**: Clone the Slang ESP32-IDF component repository into your project's `components` directory.
```bash
git clone https://github.com/gespel/Slang.git
```
